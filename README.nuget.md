# AtomUI.City

AtomUI.City is a full-stack application framework for Avalonia and AtomUI desktop applications.

The package family provides application hosting, lifecycle, modularity, routing, MVVM integration, presentation bridge, state management, event bus, localization, security, data access, plugin infrastructure, build integration, templates, testing utilities, and CLI tooling.

Use the package that matches the layer you need:

- `AtomUI.City.Core` for host, lifecycle, modularity, DI contracts, diagnostics, and threading primitives.
- `AtomUI.City.Build` for MSBuild integration and framework build conventions.
- `AtomUI.City.Cli` for `atomui city ...` developer workflows.
- `AtomUI.City.Templates` for project, module, plugin, and test templates.
- `AtomUI.City.Testing` for framework test host utilities.
