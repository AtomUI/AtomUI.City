# AtomUI.City Release Notes

## 0.1.0

### New features

- Initial framework package structure.
- Core host, lifecycle, modularity, routing, MVVM, presentation, state, event bus, localization, security, data, plugin system, build, templates, testing, and CLI foundations.

### Breaking changes

- None for the initial package line.

### Fixes

- None for the initial package line.

### Known limitations

- Public APIs are still pre-1.0 and may change before the first stable release.
- Platform integration tests are expected to expand as desktop runtime features mature.

### Migration notes

- No migration is required for the initial package line.

### Plugin API compatibility

- Plugin API compatibility starts at `1.0` for the initial package line.
