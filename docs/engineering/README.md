# 工程文档

本目录维护 AtomUI.City 仓库工程化文档。

强制规范：

- [documentation-governance.md](documentation-governance.md)：文档先行治理规范。所有模块必须有完善文档后才能开始写代码，任何改动必须先完成文档对齐并确认。

计划维护：

- `repository-structure.md`：仓库目录、源码项目、测试项目、引用项目说明。
- `build-system.md`：MSBuild props/targets、输出目录、构建约定。
- `coding-style.md`：代码风格、Rider/ReSharper 设置、换行符和格式化规则。
- `dependency-management.md`：NuGet 版本集中管理、升级流程和兼容性验证。
- `versioning.md`：版本号、包版本、模板版本策略。
- `packaging.md`：NuGet 包、模板包、插件包打包规则。
- `release-process.md`：发布流程和发布前验证。
