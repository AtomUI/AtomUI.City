# 架构决策记录

本目录维护 AtomUI.City 的 Architecture Decision Records。

ADR 用于记录已经形成约束力的重要决策，包括背景、决策、影响和后续约束。

命名规则：

```text
0001-short-title.md
0002-short-title.md
```

建议格式：

```markdown
# 0001 决策标题

状态：Accepted
日期：YYYY-MM-DD

## 背景

## 决策

## 影响

## 后续约束
```
